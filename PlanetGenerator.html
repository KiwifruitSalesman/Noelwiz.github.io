What?

The idea is that it will be able to generate planets, and stars, giving you their mass, radius, density, stuff like that. Additionally, I'm hoping to allow people to make some special planets or planetiary systems, like specifically generate a star with a habitable planet, or with a gas gient orbiting close to the star.

Current State

Currently the project is in it's early stages. Mostly Python skeletons, and planning is done. Most of the different classes for objects have fields, and some functions to calulate basic properties, but there's not yet any actual generation working, nor is that information convayed to users yet.

Other Info

Currently this project isn't a very high priority for me, so it may be a long time before it's in any useable state, but if anyone is really interested in it, tweet me or something and I'll be happy to talk about it.
