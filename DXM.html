What is this?

Think of tetris, now imagine what it would be like if gravity rotated periodically, and the blocks were only 3 squares.

Current State

Currently the game is not exactly playable. It's still in the same state as it was at the end of the game jam. You can manipulate gravity, move blocks and the blocks will move periodically, but that's about it. The biggest problem is with collision, and the grid.

Other Info

I think both me and Vae I worked with want to fix and get this game to a playable state in the future. I did the coding, while Vae did the music and art for the game.

How do I use this?

Instillation


Download the files from Itch.io above. After that, you should be able to run the game and see how it doesn't really work that well.

Controlls

All commands are triggered with $. If a meme's name has a space, it's tag must be used in any commands. Time advances every time you move, or after a second. Gravity rotates after about 5 blocks are placed, although the blocks currently don't fall down to match gravity.
